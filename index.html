Spark Server

Spark was a chat application created as a group project in a Software Engineering course. It was written in JavaScript for use with Node.
The Spark Server is an API server for a chat client, which was required to provide all information through basic requests. It is also very well documented and was almost entirely assembled solo, which makes it a great candidate for portfolio work. All API were originally documented on a team Confluence page, I am currently working towards migrating them to Markdown files to rehost on Github.

What I could do was recreate it.

Chocolate 2 is slightly less terrible wiki software made using Python for CGI scripts. It has the ability to perform a number of basic actions, such as allowing edits and formatting wiki pages. At the very least, it is a basic showcase of my understanding of Python.
